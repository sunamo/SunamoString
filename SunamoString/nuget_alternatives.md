# NuGet Alternatives to SunamoString

This document lists popular NuGet packages that provide similar functionality to SunamoString.

## Overview

String manipulation utilities

## Alternative Packages

### Humanizer
- **NuGet**: Humanizer
- **Purpose**: String humanization
- **Key Features**: Pluralize, truncate, case conversion, inflection

### System.String
- **NuGet**: System.Runtime
- **Purpose**: Built-in string methods
- **Key Features**: Split, Join, Replace, Substring, Trim

### StringExtensions
- **NuGet**: StringExtensions
- **Purpose**: String extension methods
- **Key Features**: Additional string utilities

### MoreLINQ
- **NuGet**: morelinq
- **Purpose**: Collection/string LINQ
- **Key Features**: ToDelimitedString, string joining

## Comparison Notes

Built-in string methods cover basics. Humanizer for advanced string manipulation and display.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Humanizer**: String humanization
- **System.String**: Built-in string methods
- **StringExtensions**: String extension methods
